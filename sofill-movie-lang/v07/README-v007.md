# 07 - 클래스로 데이터 타입 정의하기

## 학습 목표

- 클래스를 활용하여 여러 개의 데이터를 한 단위로 묶을 수 있다.
- 클래스와 인스턴스의 관계를 이해한다.
- 인스턴스와 레퍼런스의 관계를 이해한다.
- 레퍼런스 배열을 이해한다.
